Webflow Designer extension

Icons from top libraries, inside your canvas

IconDock helps you find, preview, and use SVG icons from many free icon sets—without leaving the Webflow Designer. Search, filter by library and style, copy clean markup, or insert on the page when you’re building.

What you can do

Search & filter

Browse thousands of icons and narrow results by library and style so you find the right glyph fast.

Copy SVG code

Copy minimal SVG markup to paste into embeds or components—sized to flex with your layout via a parent wrapper.

Insert on canvas

When you’re in the Designer, add icons into the selected container so they land where you’re already working.

PNG option

Prefer raster assets? Insert a PNG in the flow you use for images when that fits the project better.

Libraries

IconDock aggregates many popular open sets (for example Material, Font Awesome Free, Heroicons, Lucide, Phosphor, Remix, Iconoir, Bootstrap Icons, and others depending on your deployment). Always follow each icon family’s license when you ship work.
